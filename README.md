cvc# To-Do-List

To Do List App

## Project Overview

This To-Do List application allows users to manage their tasks efficiently. Users can add, edit, and delete tasks, making it easy to keep track of their daily activities.

## Features

- Mark tasks as complete

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd To-Do-List`
3. Open `index.html` in your browser.

## Usage

- Open the application in your browser to start managing your tasks.
